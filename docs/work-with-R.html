6 Working with R

R is an open source language and environment for statistical computing and graphics [1], ranked by IEEE in 2020 as the 6th most popular programming language (Python, Java, and C are the top three) [2]. If you are new to R, some of its best features, paraphrasing Wickham [3], are:

  • R is free, open source, and available on every major platform.
  • R packages provide effective tools for data analysis and visualization.
  • More than 17,750 open-source R packages are available (Jun 2021). Many are cutting-edge tools.

RStudio, an integrated development environment (IDE) for R, includes a console, editor, and tools for plotting, history, debugging, and workspace management as well as access to GitHub for collaboration and version control [4].

6.1 New to R?

This is the workshop tutorial entry point for the R beginner. Before proceeding, you should have completed

  • Install everything
  • Set up a project

Launched your workshop project (workshop.Rproj or the name you used) to start the R session

By the end of the workshop, our R beginners will have made substantive progress on three tutorials:

  • R basics A short introduction to R (in the next section)
  • Getting started: An introduction to the MIDFIELD practice data tables
  • Multiway graphs and tables: Practice working with a data structure commonly encountered when preparing graphs and tables.

6.3 After the workshop

For additional basic skills in R:

  • Graph basics
  • Data basics

To continue the midfieldr vignette sequence:

  • Case selection: Select and name a specific set of programs to explore.
  • Data sufficiency: Limited a study to students whose timely completion term is within the range of data provided by their institution.
  • Timely completion: Graduates completing their program later than their timely completion term are grouped with non-graduates for purposes of computing a persistence metric.
  • Stickiness: Compute longitudinal “stickiness,” the ratio of the number of students graduating in a program to the number of students ever enrolled in that program.
  • FYE programs: Predict the engineering program the FYE student would have declared had they not been required to enroll in FYE.
  • Graduation rate: Compute graduation rate with a more inclusive definition.
